🛒 Smart Trolley Using RFID

A Smart Trolley system designed to enhance the shopping experience by automating billing and reducing checkout time using RFID (Radio Frequency Identification) technology.
📌 Overview

This project eliminates long billing queues in supermarkets by integrating RFID tags with a smart trolley. Each product is tagged with an RFID chip, and the trolley is equipped with an RFID reader that scans items as they are added or removed. The system automatically updates the cart total in real time.
